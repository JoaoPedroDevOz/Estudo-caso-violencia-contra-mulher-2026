# Estudo-caso-violencia-contra-mulher-2026
Esse projeto abrange o estudo de caso, utilizando ferramentas python para ciência de dados, acerca dos dados coletados sobre violência contra mulher no ano de 2026. 
A ideia desse projeto não é para buscar soluções sobre como evitar esses problemas, até porque os dados não abrangem esse tipo de informação, mas sim identificar os subgrupos vítimas,
suas características e encontrar padrões nessas ocorrências para, assim, poder criar medidas protetivas e preventivas para esses grupos. 
Em resumo, é um estudo de caso para a realização de um projeto de uma disciplina da faculdade (UCB).
